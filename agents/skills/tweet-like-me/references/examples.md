# Examples

These are pattern examples, not a raw archive. Use them to learn shapes and constraints.

## `codex-teacher`

Input idea: People do not know Codex can install skills.

Variants:

```text
If you're using Codex, you might not know about skills yet

ask it to look at the skills installer and tell you what you should install

it'll scan the repo and figure out what is useful for your setup
```

```text
codex tip: skills

if codex is open right now just ask it what skills you should install

weirdly effective way to make the app feel more yours
```

## Critique Example

Input draft:

```text
We're excited to announce a revolutionary new workflow that empowers developers to unlock productivity.
```

Critique:

```text
Verdict: too corporate. no concrete workflow, no human texture.

Rewrite:
codex tip: stop announcing productivity and show the workflow

what did it do, what did you ask, where did it get weird
```
