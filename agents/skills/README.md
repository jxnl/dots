# Skills

Codex skills stored for installation to `~/.codex/skills` via `./install.sh --skills` or `./install.sh --agents`.

## Included
- `slidev-presentation-kit` - Slidev presentation workflows and references.
- `subagent` - Guidance for running non-interactive `codex exec` subagent sessions.
- `pdf-viewing` - OCR PDFs and rasterize pages to images.
- `youtube-extractor` - Extract transcripts, titles, and thumbnails from YouTube videos.
