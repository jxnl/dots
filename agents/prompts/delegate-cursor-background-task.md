# Delegate Cursor Background Task

Create a GitHub issue or Linear ticket with `@cursor` in the body to hand off work to Cursor's background agent.

## Steps

1. Detect platform: use Linear MCP if available, otherwise `gh issue create`
2. Draft ticket with:
   - Detailed task description
   - Repo name and relevant file paths
   - Links to related issues, PRs, or docs
   - Any context needed to work autonomously
3. Include `@cursor` in the body
4. Report the URL
